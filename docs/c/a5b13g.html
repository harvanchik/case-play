In(a), ifTeam Bacceptsthe illegal contact penalty, then it isTeam A's ball atA's 21and a replay fourth down. This is a double foul (Rule 10-3-3C) and the Referee should announce that both penaltiesoffset. IfTeam Bdeclinesthe penalty andTeam Aacceptsthe flag guarding penalty, then it'sTeam B's ball atB's 10.
In(b), ifTeam Bdeclinesthe illegal contact penalty, it isTeam B's ball atB's 25. IfTeam Bacceptsthe penalty, then it isTeam A's ball atA's 11and replay fourth down, double stakes.
